# ACDAN: Autonomous Cyber Defense Agent Network 🛡️🤖

**ACDAN** is an advanced, multi-agent security framework designed to detect, analyze, and mitigate network threats autonomously. It combines Deep Learning, Large Language Models (LLMs), and Reinforcement Learning into a unified defensive pipeline.

## 🚀 The Architecture
The system is divided into three specialized "Agents":

1.  **Detection Agent (ML):** Uses PyTorch-based Deep Neural Networks and Transformers to classify network traffic anomalies in real-time.
2.  **Reasoning Agent (LLM):** Leverages LLMs (Mistral/Ollama) to perform deep contextual analysis and map threats to MITRE ATT&CK tactics.
3.  **Response Agent (RL):** A Reinforcement Learning agent that learns optimal mitigation strategies (Block, Rate-limit, Quarantine) through environmental feedback.
